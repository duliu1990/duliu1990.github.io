Spring Boot - Hello Wolrd (Thymeleaf)

Author: Darren Du Date: July 24, 2017  0:14:09 Category: Java Framework Spring Boot

本例以Windows, JDK1.8, Spring Boot (1.5.4) 为基础，演示使用Spring Boot 创建Hello World项目。

使用到的技术

  1. Spring Boot 1.5.4.RELEASE
  2. Maven 3.10
  3. Java 8
  4. IntelliJ IDEA
