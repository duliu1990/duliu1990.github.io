Spring Boot + Spring Data JPA + MySQL Demo

Author: Darren Du Date: July 24, 2017  23:38:37 Category: Java Framework Spring Boot

本例以Windows, JDK1.8, Spring Boot (1.5.4) 为基础，演示使用Spring Boot + Spring Data JPA + MySQL 实现常用的增删查改操作。

以学生实体为例,来完成如下操作:

  1. 添加学生对象
  2. 查询所有学生和指定学生
  3. 更新指定的学生
  4. 删除指定的学生

用到的技术和工具

  1. Spring Boot 1.5.4.RELEASE
  2. Maven 3.10
  3. Java 8
  4. IntelliJ IDEA

项目结构

Maven pom.xml

File : pom.xml

Configure 配置

配置数据库的连接信息, 本来使用.yaml文件格式来完成配置.这种文件的语法更加强大,比.properties配置更加容易阅读和使用, Spring Boot 默认会查找application.yaml的文件.
指定当前环境使用的配置文件.
File : application.yaml

配上数据库连接信息,指定Server端口.
File : application-dev.yaml

File : Student.java

Repository 数据库操作

本例主要继承和使用Spring JPA的常用操作.

Service
