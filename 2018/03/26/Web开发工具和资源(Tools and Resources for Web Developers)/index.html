Web开发工具和资源(Tools and Resources for Web Developers)

Author: Darren Du Date: March 26, 2018  11:56:47 Category: Web开发工具和资源

设计 Design

  • InVision -InVision使团队可以自由设计，审查和用户测试产品 - 所有这些都不需要一行代码。 通过直观的原型设计，任务管理和版本控制工具，整个设计过程都在一个地方。
  • Creative Tim - Creative Tim提供基于Bootstrap的设计元素，加速您的开发工作。 从插件到复杂的工具包，他们拥有您需要的一切。
  • 谷歌字体 - 谷歌字体帮助您使网页更美观，快速，并通过伟大的排版打开。
  • Unsplash - Unsplash是找到优质且免版权的摄影的理想场所。
  • Colordot - Colordot是人类的颜色选择器。 它直观，快速，美观。
  • Dribbble - Dribbble是一个展示和宣传，推广，发现和探索设计的地方。
  • ByPeople - ByPeople.com是一个有用的资源（免费赠品，工具评论和代码片段）的内容网络。
  • Bootstrap Studio - 一个功能强大的桌面应用程序，用于使用Bootstrap框架创建响应式网站。
  • Canva - 使用Canva的拖放功能和专业布局设计始终令人惊叹的图形。
  • Pikwizard - 一个免费的股票照片网站。

协作工具 Collaboration Tools

  • Slack - Slack将您的所有通讯集中在一个地方。 它是实时消息传递，归档和搜索现代团队。
  • Trello - Trello是管理项目和组织任何事情的简单，自由，灵活和可视化的方式。
  • RingCentral - RingCentral是最先进的云通信解决方案的供应商，可帮助现代公司以他们所需的方式工作。
  • Asana - Asana是团队跟踪他们工作并获得结果的最简单方式。
  • HipChat - 群聊，视频聊天和屏幕共享。
  • GitHub - GitHub是一个受你工作方式启发的开发平台。 您可以托管和查看代码，管理项目，并与数百万其他开发人员一起构建软件。
  • Bitbucket - Bitbucket是专业团队的Git解决方案。
  • - Team@OSCTeam@OSC 是一个团队协作开发平台,轻松管理轻量级团队。代码运行平台(PaaS)、代码质量检查应有尽有。

测试和展示 Testing and Showcasing

  • jsfiddle - JSFiddle是一个在线社区，用于测试和展示用户创建和合作的HTML，CSS和JavaScript代码片段，称为“小提琴”。 它允许模拟AJAX调用。

学习资源 Learning Resources

  • freeCodeCamp - 帮助您学习编码的开源社区。
  • Codecademy - Codecademy是我们团队学习，教授和创造未来在线学习体验的最佳场所。
  • - Dreamweaver Tutorial 使用Dreamweaver创建网站的分步指南。
  • CodeSchool - 有抱负和有经验的开发人员的互动学习目的地。
  • Tutorialzine - 一个有才华的开发者共同学习的社区。
  • Bento - 顶级工程师提供免费的完整堆栈网络开发课程。
  • w3schools - W3Schools是一个Web开发人员网站，提供有关Web开发语言（如HTML，CSS，JavaScript，PHP，SQL，W3.CSS和Bootstrap）的教程和参考资料，涵盖了Web编程的大多数方面。

前端框架 Front-end Frameworks

  • Bootstrap - Bootstrap是用于在Web上开发响应式移动第一项目的最流行的HTML，CSS和JS框架。
  • Semantic UI – 语义是一种开发框架，可帮助使用人性化的HTML创建漂亮，响应式布局。
  • UIkit - 用于开发快速而强大的Web界面的轻量级和模块化前端框架。
  • Foundation – Foundation是一个响应式前端框架系列，可以轻松设计美观的响应式网站，应用程序和电子邮件，在任何设备上看起来都非常棒。

文本编辑器 Text Editors

  • Brackets – 一个理解网页设计的现代化开源文本编辑器。
  • Coda2 – Coda是您在一个漂亮的应用程序中手动编写网站所需的一切。
  • Visual Studio Code - Visual Studio Code是一个简化的代码编辑器，支持诸如调试，任务运行和版本控制等开发操作。
  • Sublime Text – Sublime Text是用于代码，标记和散文的复杂文本编辑器。
  • Atom – Atom是一个使用HTML，JavaScript，CSS和Node.js集成构建的桌面应用程序。
  • WebStorm - 强大的IDE用于现代JavaScript开发。

图标 Icons

  • Swifticons – Swifticons是一套优质的2,264个高质量图标，覆盖了6种可编辑风格的20个类别。
  • Icons8 – 52,600个免费平面图标，任何格式，大小和颜色均可在20秒内完成。
  • Nucleo App - Nucleo是一个拥有超级权力的图标库。
  • Iconfinder - Iconfinder为数百万创意专业人士提供高质量的图标。
  • Font Awesome – Font Awesome为您提供可立即定制的可缩放矢量图标 - 大小，颜色，阴影以及任何可以使用CSS强大功能的图标。
