Author: Darren Du Date: May 7, 2018  19:33:53 Category: Java Framework Spring Boot

在Spring Boot中来发送邮件是很方便的事情，只需要添加对应的依赖jar,然后配置邮箱服务器地址和账号就可以实现邮件的发送。本示例使用JDK1.8, Spring Boot 1.5.9.RELEASE来演示邮件的发送。
